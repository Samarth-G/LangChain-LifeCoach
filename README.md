# LangChain-LifeCoach
Bollywood inspired Life Coach built using LangChain and OpenAI in python (running on Flask)

<img src="https://github.com/Samarth-G/LangChain-LifeCoach/assets/18361189/49a5d869-9cc4-4222-bd6a-01ba3a8dd52e" alt="mainpage" width="250"/>

Steps:
1. Clone repository
2. Create env with environment.yml
3. Export OPENAI_API_KEY in env (`export OPENAI_API_KEY = "Enter_API_key_here"`)
4. Run `python app.py` in the root folder
5. Open http://127.0.0.1:5000 on your browser
